HTML is a markup language that defined the meaning and structure of a web document.

HTML is the standard language used to describe the meaning and structure of a web document. It was born out of SGML, so as its sibling XML, they both are markup language that specify strict rules to organise data and share without misinterpretation. The main differences are: HTML focus on how the data should look on the website/web document, and it has a fix sets of tags; while XML focus more on transport the information itself, at the same time preserve its meaning, developers are welcome to invent new tag for XML.

Hypertext made the web powerful by linking extra resources and within a page. HTML and HTTP together form the foundation of the world wide web, one for creation, another for delivery. The first HTML document was officially published by Tim Berners-Lee back in 1991. It's used to give a quick overlook about WWW and usage of HTML. It looks quite simple, yet perfectly demonstrate the usefulness of HTML with help of hyperlink- to share and organise web resources. The link can be find in here.

HTML has certainly evolved a lot since its creation in 1989, when only few element and limited styling option are available, at the time websites look rather dull apart from some text. The latest version to date is HTML 5 which made standard by W3C in 2014. It supports most of the modern browser, graphic canvas that allow drawing using Javascript, multimedia such as video and audio element, charset attribute in meta tag that defines the standard character encoding, self-explained layout tags such as <nav> tag and <article> tag.
