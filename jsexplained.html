Javascript has this controversial reputation due to some of the weird syntax, but it\'s undoubtedly one of the most widely used web tech nowadays.

There is no Javascript in the earlier stage of the web ecosystem, and suddenly people realised they want something that makes the website more dynamic, that leads to the creation of Javascript. The initial intention was to make it based on well-known language Java and C.

In 1995, Brendan Eich put together Javascript core, at the time called Mocha, expand the web browser function further.
It then been renamed and standardised by ECMA (European Computer Manufacturer Association), nowadays the regularly referred ES8 for example means ECMAScript version 8, it is the guideline for implement Javascript.

More associated technology such as JQuery, Chrome browser V8 engine, NodeJS and some more frameworks emerged post 2000s and broaden the task which JS can do.

Not this AJAX...

AJAX stands for Asynchronous JavaScript and XML(despite the fact that nowadays we use JSON more). It\'s a technique that allows user to send a request and carry on, and deal with the response later on when have one.

With AJAX, users no longer need to constantly do reload to see the update.

By combining AJAX technique and third party API, web developers no longer have to worry about the data source or data storage (well.. in some scenarios) It is as simple as making query to a url address.

Example: try to search your favourite movie below

In the previous simple AJAX demonstration, the code under the hood contains these steps:

  1. Set up the search bar and submit button in plain HTML, also create an empty div for later on display the result returned
  2. Attach a Javascript function to the onclick event on the button
  3. Upon click, send request to The Movie Database API
  4. Convert the response to JSON (Javascript Object Notation) for easy access
  5. Extract information from the result array, create HTML to accommodate the information
  6. Append the element to the div prepared beforehand, this step uses Javascript to modify DOM structure
  7. TA-DAA!! There it comes a simple movie search engine :D
